# Teacher Assistance

A Flask-based classroom support app for lesson planning and student assistance workflows.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python app.py
```

## Project Structure

- `app.py` contains the Flask application
- `templates/` contains the HTML templates
- `static/` contains static assets
- `data/` contains local data files
